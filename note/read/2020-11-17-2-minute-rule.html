Get started with 2-minute rule

Whenever you find it hard to started on a task, consider scaling it down into a 2-minute version. For example

当你觉得一个任务很难开始的时候，考虑将其缩略为一个 2 分钟版本的任务，例如：

  • Read a book → Read on page 读一本书 → 读一页书
  • Write an essay → Write one sentence 写一篇文章 → 写一个句子
  • Run 10KM → Wear my running shoes 去跑 10 公里 → 穿上运动鞋
  • Do 100 push-ups → Do 1 push-up 做一百个俯卧撑 → 做一个俯卧撑
  • Eat more vegetables → Eat an apple 多吃水果蔬菜 → 吃个苹果
  • Study for interview → Skim through my notes 准备面试 → 瞄一眼自己的笔记
  • Build a program → Code a function/line 写一个程序 → 写一行代码/一个函数

The idea is to make it super easy to get started. Once you pass the starting point, which is arguably the hardest step, you start to gain momentum to keep doing the task itself.

这里的核心概念是让任务变的超级容易开始，任务最艰难的步骤可能就是开始了，一旦你开始了，你可能就获得了持续下去的动力了。

  • Read 1 page → Read 10 pages → Read 1 chapter 阅读一页 → 阅读 10 页 → 阅读一章
  • Write 1 sentence → Write 1 opening paragraph → Write the body 写一个句子 → 写一个开篇断了 → 完成文章主体
  • Wear my running shoes → Walk for 5 minutes → Run for 5 minutes 穿上运动鞋 → 走 5 分钟 → 跑 5 分钟

As you can see, once you start, it's much easier to continue doing it. Sometimes, you'll find yourself completing the task even before you even notice it.

是不是一旦开始，其实就挺容易坚持下去了，有事甚至你都没注意到你已经轻松的完成了任务。

个人生词：

  • push-up 俯卧撑
  • skim 粗略浏览，扫一眼
  • arguably 可能
  • momentum 动力、冲量，动量
